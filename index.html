Kilo

This is my C++ port of antirez's (Salvatore Sanfilippo) kilo, which is a small text editor in less than 1000 lines of C code. It does not depend on any library and uses standard VT100 escape sequences. This project is currently a work in progress.

Mini-Message-Board

A miniature message board application, wherein you can create and view messages. Each message consists of the message text, its author, and the time it was created. It is written in HTML, CSS, and TypeScript.

Calculator

This is a calculator application written in HTML, CSS, and TypeScript. It makes use of the Shunting Yard algorithm to parse and evaluate arithmetic expressions to yield the expected result.
